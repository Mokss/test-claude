# Платформа для заданий учеников

## Функциональные требования

- Даем список задач
- Даем возможность открыть отдельно задачу
- Даем возможность отправить решение задачи

- Ролевая модель Учитель/Ученик

- Учитель
    - Список Учеников
    - Список решенных задач ученика
    - Просмотр самого решения задачи ученика
    - Оценка и комментарии по решению ученика
    - Уведомление что ученик сделал задание
    - Возможность загрузить решение

- Сбор Статистики
    - Скорость исполенния задание
    - Колличество ошибок
    - Качество



- Сандбокс для запуска кода
-- Запустить код
-- Должен уметь отдавать stdout/stderr
-- Должен отдавать информацию о пройденных тестах
-- Загрузка файлов

-- Не должен выйти за пределы своей изоляции
-- Не должен зависнуть намертво
